# 小説「がくせいになりました」

セドリックは異世界と思われるグンマー王国で前世の記憶を取り戻した。猫のトイレのような排泄場所、なぜか通じる日本語、読めるけど書けない言語体系、負の数が存在しない数学…そんな中、衛生環境などに取り組み「がくせい」となるファンタジー小説。
